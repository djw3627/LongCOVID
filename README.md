# Dataset Derived from Household Pulse Survey

### Introduction

The dataset was derived from seven microdata files from HPS data collection phases 3.5 to 3.7, spanning the timeframe between June 1, 2022 and December 19, 2022. The original microdata files containing individual responses to the Household Pulse Survey are available for download fromÂ <https://www.census.gov/programs-surveys/household-pulse-survey/data/datasets.html>
